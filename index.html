Computer Parts

Network card A network card connects the computer to the internet. It typically connects to a LAN network. It is connected by a PCI Express slot.
RAM RAM or random access memory is a small and fast storage for storing memory that needs to be accessed fast. All the memory in the RAM gets deleted when the power gets turned off.
Hard Drive Hard drive stores pernament data. The computer accesses it by a SATA cable.
Video Card Video card can generate graphical images like a video card, but can also display those images by an HDMI, DisplayPort or VGA.
Graphics Card Graphics card generates graphical images and stores it in VRAM. It connects by a PCI Express Slot.
